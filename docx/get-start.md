# Get Start

## 安装

::: code-group

```bash [npm]
npm install @rika_/utils
```

```bash [pnpm]
pnpm install @rika_/utils
```

```bash [yarn]
yarn add @rika_/utils
```

```bash [bun]
bun install @rika_/utils
```

:::

## 最简使用
```ts
import { jsonToFormData } from '@rika_/utils'

const formData = jsonToFormData({ id: 1 })
```
